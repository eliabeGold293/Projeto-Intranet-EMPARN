# Classes de Botões do Bootstrap 5

Abaixo estão as principais classes que você pode usar para criar botões no Bootstrap 5, com explicações claras e objetivas.

---

## **1. Botões Padrão (Cores principais)**

### `.btn`

Classe base obrigatória para qualquer botão Bootstrap. Deve sempre vir junto com uma classe de estilo.

### `.btn-primary`

Botão azul padrão — usado como ação principal em uma tela.

### `.btn-secondary`

Botão cinza — usado como ação secundária.

### `.btn-success`

Botão verde — geralmente usado para ações positivas (salvar, confirmar, ok).

### `.btn-danger`

Botão vermelho — usado para excluir, remover ou ações destrutivas.

### `.btn-warning`

Botão amarelo — usado para avisos.

### `.btn-info`

Botão azul claro — usado para informações adicionais.

### `.btn-light`

Botão branco/cinza claro — combina bem com temas escuros.

### `.btn-dark`

Botão cinza escuro — combina com layouts claros.

### `.btn-link`

Botão que parece um link — usado quando a ação não é muito relevante.

---

## **2. Tamanhos dos botões**

### `.btn-lg`

Botão grande.

### `.btn-sm`

Botão pequeno.

(Quando não usa nenhum tamanho, o botão fica no tamanho padrão.)

---

## **3. Estados dos botões**

### `.active`

Indica que o botão está ativo.

### `.disabled`

Desabilita o botão visualmente (mas ainda permite clique via HTML, se não usar atributo `disabled`).

### Atributo `disabled`

Desabilita totalmente o botão, impedindo clique.

```html
<button class="btn btn-primary" disabled>Salvar</button>
```

---

## **4. Botões com contorno (outline)**

São versões "vazadas" dos botões padrões.

### `.btn-outline-primary`

### `.btn-outline-secondary`

### `.btn-outline-success`

### `.btn-outline-danger`

### `.btn-outline-warning`

### `.btn-outline-info`

### `.btn-outline-light`

### `.btn-outline-dark`

Ideais para botões com menos destaque no design.

---

## **5. Botões com largura total**

### `.w-100`

Expande o botão para ocupar 100% da largura do container.

```html
<button class="btn btn-primary w-100">Entrar</button>
```

---

## **6. Botões com ícones (Bootstrap Icons)**

```html
<button class="btn btn-primary">
    <i class="bi bi-plus"></i> Adicionar
</button>
```

Você pode usar qualquer ícone dentro do botão.

---

## **7. Botões em grupo**

### `.btn-group`

Agrupa botões lado a lado.

```html
<div class="btn-group">
    <button class="btn btn-primary">Esquerda</button>
    <button class="btn btn-primary">Meio</button>
    <button class="btn btn-primary">Direita</button>
</div>
```

### `.btn-group-vertical`

Grupo vertical.

---

## **Resumo rápido**

**Sintaxe base:**

```
<button class="btn btn-primary">Texto</button>
```

**Partes principais:**

* `btn` → classe obrigatória
* `btn-{cor}` → define o estilo
* `btn-{tamanho}` → define o tamanho (opcional)
* `btn-outline-{cor}` → versão vazada
* `w-100` → largura total
* `disabled` → desabilitado

Se quiser, posso gerar um **arquivo HTML demonstrativo** com todos esses botões exibidos visualmente.
